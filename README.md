# What Comes Next?

**Tiny Transformers Book 1**

An interactive board book teaching prediction — the fundamental idea behind language models.

Babies learn that patterns help us guess what comes next. This IS what a language model does.

## The Series

| Book | Title | Core Concept |
|------|-------|--------------|
| 1 | What Comes Next? | Prediction |
| 2 | Look Here! | Attention |
| 3 | Watch and Learn! | Few-shot Learning |
| 4 | Building Blocks | Tokenization |
| 5 | Maybe This, Maybe That! | Probability |
| 6 | I Remember! | Memory |
| 7 | Bigger and Bigger! | Scaling |
| 8 | Oops, Try Again! | Learning from Mistakes |
| 9 | Better Together! | MoE |
| 10 | Good Job! | Reinforcement Learning |
